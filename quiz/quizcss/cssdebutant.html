Quiz

1.Que signifie CSS ?

a) Cascading Style Sheets

b) Create Simple Samples

c) Choucroute et Saucisses d'Alsace


2. A quoi sert le langage CSS ?

a) A réaliser des pages dynamiques

b) A ajouter du style aux documents web

c) A insérer du contenu dans une page internet


3. Pourquoi utilise-t-on généralement du CSS ?

a) Pour se compliquer la vie bien sûr

b) Pour séparer le contenu et la présentation des documents web

c) Cela permet de faire de plus jolis dégradés de couleurs


4. Où est-il conseillé de placer le code CSS?

a) Dans le body, c'te question

b) Entre les balises head, c'est bien plus propre

c) Dans un fichier externe utilisable pour plusieurs pages


5. Dans quoi définit-on les déclarations CSS ?

a) Entre deux crochets séparées par des tirets [ ... - ... ]

b) Entre deux guillemets séparées par des slashs " ... / ... "

c) Entre deux accolades séparés par des points-virgules {... ; ...}


6. Dans la règle suivante, quel élément deviendra rouge ? strong#intro {background-color: red;}

a) L'élément dont l'id est "intro" et qui est contenu dans un élément strong

b) L'élément "strong"" dont l'id est "intro"

c) Tous les éléments ""strong" contenus dans un élément dont l'id est "intro"


7. Quel élément sépare la propriété de sa valeur ?

a) Le signe espace ( )

b) Le signe égal (=)

c) Le signe deux points (:)


8. Pourquoi est-il d'usage de sauter une ligne entre chaque déclaration ?

a) Pour rien du tout

b) Pour que le code soit bien interprété par les navigateurs

c) Pour rendre le code plus lisible et la maintenance plus facile


9.A quoi sert le sélecteur de classe : h2.nomDeClasse ?

a) Ah bon ... C'est censé servir à quelque chose ?

b) A créer une ancre sur la balise h2

c) A définir différentes propriétés pour l'élément h2 dont la classe est "nomDeClasse".


10. Combien de règles peut contenir un fichier CSS ?

a) Il n'y a pas de limite, on peut en créer autant que l'on veut

b) Si le poids du fichier n'excède pas 20Ko, c'est tout bon

c) 150
